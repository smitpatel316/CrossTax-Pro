# Cross-Border Tax Filing System

## Project Overview

**Project Name:** CrossTax Pro  
**Type:** Web Application (SaaS)  
**Core Functionality:** Automated tax preparation software for US-Canada cross-border individuals (expats, remote workers, immigrants, snowbirds)  
**Target Users:** US citizens in Canada, Canadian citizens in US, dual citizens, cross-border workers

---

## Key Features

1. **Multi-Jurisdiction Income Tracking**
   - US Form 1040 support
   - Canadian T1 return support
   - FBAR (Foreign Bank Account Reporting)
   - Form 5471 / Form 8865 for foreign entities

2. **Tax Treaty Optimization**
   - US-Canada Tax Treaty article application
   - Foreign Tax Credit optimization
   - Totalization agreement handling (social security)

3. **Automated Calculations**
   - Currency conversion (historical rates)
   - Tax bracket mapping for both countries
   - Carryforward/carryback calculations

4. **Document Management**
   - W-2, T4 import
   - Receipt storage
   - Export to PDF

5. **CPA-Level Validation**
   - Rule engine with CPA expertise built-in
   - Cross-reference checks between US/Canada filings
   - Audit trail

---

## Technical Requirements

- **Frontend:** React + TypeScript
- **Backend:** Node.js + PostgreSQL
- **Tax Engine:** Modular rule system (CPA-reviewed)
- **Security:** Bank-grade encryption, SOC2 compliant

---

## Success Metrics

- 100% accuracy on tax treaty calculations
- Audit-ready documentation
- < 30 min to complete filing
