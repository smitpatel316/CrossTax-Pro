## Key UI Principles

1. **Progress always visible** - Show step X of Y
2. **Autosave** - Never lose data
3. **Calculator always live** - Show tax impact as they enter
4. **Explain jargon** - Tooltips on every tax term
5. **Error prevention** - Validate before submission
6. **Mobile-first** - Filers want to work on phone/tablet
